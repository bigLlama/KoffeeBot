KoffeeBot Commands - Moderation

Moderation

  • /clear - Deletes a specified amount of chat messages
  • /addrole - Give someone a role from your server
  • /removerole - Remove a role from someone
  • /addnote - Create notes about a server member
  • /editnote - Edit an existing member's notes
  • /notes - View a server member's notes
  • /inspect - Inspect a server member
